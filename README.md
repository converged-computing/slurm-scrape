# Slurm Scrape

This is a simple scrape of SLURM stuff (starting with documentation pages into json structures) for further TBA usage.

## Usage

Setup environment!

```bash
python -m venv env
source env/bin/activate
pip install -r requirements.txt 
```

And scrape

```bash
cd ./docs
python scrape.py
```
